Configuration Settings

Valid configurations:

  • Set index page. "Index" SP (name)
  • Set 404 page. "404" SP (name)
  • Set 301 redirects (if a directory without '/' at end, then automatically redirect with a '/' at end. This allows the href to be relative to the directory). ("301" SP (srcName) SP (destName))*
  • Set whether paths are case sensitive or not. "CaseSensitivePaths" SP (true/false)
  • Set max connections in backlog. "Backlog" SP (number)
  • Set whether a request for a directory shows the directory listings or a 404 page. "DirListings" SP (true/false)
